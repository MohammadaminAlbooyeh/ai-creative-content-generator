# Prompt Engineering

## System Prompts

System prompts define the AI's role and behavior. Each generator type has optimized system prompts.

## Prompt Structure

- **System Prompt**: Defines the AI's role
- **User Prompt**: The specific generation request
- **Parameters**: Temperature, max tokens, etc.

## Best Practices

1. Be specific about the desired output format
2. Include relevant context and constraints
3. Specify tone, style, and audience
4. Use examples for complex tasks
5. Iterate and refine based on results
